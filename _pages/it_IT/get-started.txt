### Lettura necessaria

Console con versioni di sistema differenti richiedono procedure differenti per installare un Custom Firmware. In questa pagina potrai trovare il corretto punto di partenza per la tua console.

Prima di iniziare, gli utenti Windows dovrebbero attivare l'opzione per visualizzare le estensioni dei file. Basta seguire le istruzioni indicate nella pagina delle [Estensioni dei file (Windows)](file-extensions-(windows))!
{: .notice--info}

Per proseguire, clicca sul collegamento corrispondente alla tua versione di sistema dalla tabella sottostante. I campi "da" e "a" sono inclusivi. This means that, for example, the "from 1.03 to 3.57" row includes 1.03, 3.57, and all versions in-between.

Puoi scoprire la versione firmware della tua console in "Impostazioni">"Sistema">"Informazioni sul sistema".
