### Lettura necessaria

La catena di exploit h-encore per PS Vita (TV) permette l'installazione di applicazioni homebrew sulla tua schermata home. È compatibile con le versioni firmware da 3.65 a 3.73.

Ricorda che l'exploit h-encore non è "permanente" (cioè non rimane installato dopo che la console viene riavviata). Questo significa che dovrai eseguire nuovamente l'exploit dopo ogni riavvio.

Oltre ad installare l'exploit h-encore, abiliteremo l'accesso a homebrew "non sicuri" che danno permessi aggiuntivi alle applicazioni homebrew. È un concetto analogo alla modalità "amministratore" sui computer.

Se sei su versioni firmware dalla 3.65 alla 3.68, userai l'exploit h-encore, se sei su versioni firmware dalla 3.69 alla 3.73, userai invece l'exploit h-encore². Il programma che useremo per installare l'exploit sulla tua console (finalhe) sceglierà l'exploit corretto per te.

Se hai la PS Vita 1000, devi avere una memory card ufficiale Sony (di qualunque dimensione) per seguire questa guida. Questa restrizione non vale per la PS Vita 2000 e la PS TV, poiché queste console hanno già una Memory Card integrata.
{: .notice--info}

### Cosa Ti Serve

* L'ultima versione di [finalhe](https://github.com/soarqin/finalhe/releases/latest)
  - Se sei su un sistema operativo MacOS o Linux dovrai compilare finalhe tu stesso
* The latest release of [VitaDeploy](https://github.com/SKGleba/VitaDeploy/releases/latest)
  - Donwnload the `VitaDeploy-FHE.zip` file

### Istruzioni

#### Sezione I - finalhe

1. Copy the contents of the finalhe `.zip` file to a folder on your computer
1. Copy the VitaDeploy `.zip` file inside the finalhe folder
  + Do not extract this file
1. Avvia finalhe sul tuo computer
  + Se stai usando un computer Windows e ti viene chiesto di concedere l'accesso di rete a finalhe tramite il firewall, fallo
1. Avvia l'applicazione Gestione Contenuto sulla tua console
1. Tap "Copy Content"
  + If your device is already connected to your computer via USB, you can skip to [Section II - h-encore Transfer](#section-ii---h-encore-transfer)
  + If your device is not detected over USB, install [QcmaDriver_winusb](https://github.com/soarqin/finalhe/releases/download/v1.91/QcmaDriver_winusb.exe) and try again
1. Tap "PC"
1. Seleziona il metodo che preferisci per connetterti a finalhe
  + Se ti viene chiesto di accedere con un account PlayStation Network, fallo
  +Se non hai un account Playstation Network, creane uno
1. Select or register your computer if prompted
  + Se ti appare un messaggio che ti chiede di effettuare un'aggiornamento, riavvia la console e riprova
  + If you still encounter this message, enable Flight Mode in the System Settings and try again (this will *not* work on PS TVs)
  + If you *still* encounter this message, update to the latest firmware version and try again

#### Sezione II - Trasferimento di h-encore

1. Select "Trim h-encore to ~7MB"
1. Click on the small arrow on the right-hand side of the finalhe window
  + This can be hard to spot, especially on high-DPI displays
1. Tick the "VitaDeploy" box
1. Click "Let's GO!"
  + I file dell'exploit saranno scaricati automaticamente e preparati
  + Questo passaggio richiederà un po' di tempo
1. Tap "PC -> PS Vita System" or "PC -> PS TV System" on your device
1. Tap "Applications"
1. Tap "PS Vita"
1. Seleziona "h-encore" o "h-encore²" in base a quale viene visualizzato
1. Select "VitaDeploy"
1. Tap "Copy"
1. Tap "OK"
  + The h-encore exploit and VitaDeploy application will be copied to your device
  + Questo passaggio richiederà un po' di tempo
1. Chiudi l'applicazione Gestione Contenuto sulla tua console
1. Chiudi finalhe sul tuo computer

#### Sezione III - Configurazione di h-encore

1. Launch the h-encore(²) application
  + If prompted about trophies, select "Yes"
1. Se l'exploit ha avuto successo, ti ritroverai sull'h-encore bootstrap menu
  + If the exploit gets stuck, reboot your device and try again
  + If you can't reboot normally, hold the power button down for over 30 seconds to force a reboot
1. Press (Cross) to exit the application

#### Sezione IV - Configurazione di HENkaku

1. Apri le Impostazioni
1. Apri `Impostazioni HENkaku`
  + Se non trovi Impostazioni HENkaku, seleziona "Reset taiHEN config.txt" nell'h-encore bootstrap menu, quindi riprova
1. Spunta "Abilita Homebrews non sicuri"
1. Ritorna a Impostazioni HENkaku
1. Chiudi le impostazioni

#### Section V - Installing essential homebrew

1. Launch the VitaDeploy application
1. Tap "App downloader"
  + If you don't see a green menu with this option, make sure you've [enabled unsafe homebrew](installing-h-encore#section-iv---configuring-henkaku)
1. Select "[VitaShell](https://github.com/TheOfficialFloW/VitaShell/releases/latest/) *(A powerful file manager)*""
1. Tap "Download the selected apps"
1. Wait for the app to finish downloading
  + When finished, you should be presented with the `VitaShell.vpk` file
1. Press (Cross) on the `VitaShell.vpk` file to install it
1. Premi (Croce) per confermare l'installazione
1. Press (Cross) to confirm the install again when prompted
