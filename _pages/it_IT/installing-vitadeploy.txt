---
title: "Installing VitaDeploy"
---

{% include toc title="Indice" %}

The guide uses the VitaDeploy application as the primary method of installing other apps and a custom firmware. If you didn't install VitaDeploy when installing h-encore, you can install it yourself with molecularShell or VitaShell.

Per installare i file `.vpk` (pacchetto contenuti) necessari sulla tua console, useremo il [File Transfer Protocol (FTP)](https://wikipedia.org/wiki/File_Transfer_Protocol) per copiare i file sulla tua memory card.

### Cosa Ti Serve

* Un client FTP come [WinSCP](https://winscp.net/) o [CyberDuck](https://cyberduck.io/)
  + In alternativa, puoi anche utilizzare la funzionalità di trasferimento USB di VitaShell
* The latest version of [VitaDeploy](https://github.com/SKGleba/VitaDeploy/releases/latest/)
  + Download the `VitaDeploy.vpk` file

#### Sezione I - Preparazione

1. Avvia VitaShell
1. Premi (Start) per aprire le impostazioni di VitaShell
1. Premi (Croce) su "SELECT button" per cambiare la modalità a "FTP"
  + In alternativa, puoi lasciare l'impostazione a "USB" e trasferire tutti i file tramite un cavo USB
1. Premi (Cerchio) per chiudere le impostazioni di VitaShell
1. Premi (Select) per abilitare l'accesso FTP sulla tua console
1. Apri il tuo client FTP sul tuo computer
1. Immetti l'indirizzo IP e la porta visualizzati sulla console
1. Utilizzando il tuo client FTP, entra in `ux0:`-> `data/`
1. Transfer `VitaHBBrowser.vpk` to the `data/` folder
1. Premi (Cerchio) sulla tua console per chiudere la connessione FTP

#### Section II - Installing the application

1. Avvia VitaShell
1. Navigate to `ux0:` -> `data/`
1. Press (Cross) on "VitaDeploy.vpk"
1. Premi (Croce) per confermare l'installazione
1. Press (Cross) to confirm the install again when prompted

VitaDeploy should now be installed to your home screen.
