### Lecture requise

Des consoles de versions différentes requièrent des étapes différentes pour atteindre l'objectif final qu'est le Custom Firmware. Cette page vous aidera à trouver par où commencer en fonction de votre console.

Avant de commencer, les utilisateurs de Windows devraient permettre d'afficher les extensions de fichiers suivant ce guide [Extensions de fichiers (Windows)](file-extensions-(windows)) !
{: .notice--info}

Sélectionnez la page appropriée à votre version dans le tableau ci-dessous. Notez que les champs "De" et "À" sont inclusifs. This means that, for example, the "from 1.03 to 3.57" row includes 1.03, 3.57, and all versions in-between.

La version de votre console peut être trouvée sous le menu Info système de la console, dans la catégorie Système de l'application Paramètres de la console.
