### 必讀事項

h-encore 是一系列專門用於 PS Vita (TV) 的漏洞；此漏洞將允許您安裝自製軟體至您的主畫面中。 它與韌體版本 3.65、3.67、3.68 相容。

請注意，h-encore 系列漏洞並非「永久性」的 (也就是說該漏洞將於主機重啟後失效)。 好消息是待會持有韌體版本 3.65 的主機可以安裝 HENkaku Ensō 來解決這問題。 For devices on 3.67 or 3.68, you will, unfortunately, need to perform a small portion of these instructions (which will be indicated) after every reboot until a new persistent exploit is developed.

除了安裝 h-encore 漏洞外，我們也將啟用『非安全』的自製程式使用權以提高自製程式的執行權限。 This idea could be considered analogous to the "administrator" mode on a computer.

不幸的是，h-encore 與先前的 PS Vita (TV) 漏洞相比起來較難安裝。 For Windows users, there is a tool which automates some of the processes, but Linux and Mac users will need to perform the entire installation manually.

如果您使用的是 PS Vita 1000，那麼您必須要有一張原廠 Sony 記憶卡 (任意大小) 才能繼續。 此限制不適用於 PS Vita 2000 或 PS TV，因為這些主機皆有內建記憶卡/內存。
{: .notice--info}

### 方法

___

#### 安裝 h-encore (透過 Windows 自動安裝)

此方法需要一台裝有 Windows 的電腦。
    
透過『auto h-encore』來自動安裝部分的 h-encore 設置。

繼續至[安裝 h-encore (透過 Windows 自動安裝)](installing-h-encore-(windows-auto))
{: .notice--primary}

___

#### 安裝 h-encore (透過 Windows 手動安裝)

此方法需要一台裝有 Windows 的電腦。
    
The h-encore installation is performed manually and requires the use of the command line.

繼續至 [安裝 h-encore (透過 Windows 手動安裝)](installing-h-encore-(windows-manual))
{: .notice--primary}

___

#### 安裝 h-encore (透過 Linux / Mac)

此方法需要一台裝有 Linux 的電腦或者一台 Mac。
    
The h-encore installation is performed manually and requires the use of the command line.

繼續至 [安裝 h-encore (透過 Linux / Mac)](installing-h-encore-(linux-mac))
{: .notice--primary}
