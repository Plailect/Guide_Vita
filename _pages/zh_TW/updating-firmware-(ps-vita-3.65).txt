### 必讀事項

The h-encore exploit is only compatible with firmware versions 3.65 and above. 因此，其他韌體必須先透過一項自製軟體更新才能使用本漏洞。

While the h-encore exploit itself is compatible with higher firmware versions, users on a firmware version under 3.65 should only update to firmware version 3.65 as it is the latest version which still supports the Ensō boot-time exploit.

請注意，您的主機僅能透過該軟體更新，而非降級。 If you are on a firmware version higher than 3.65, you should return to [Get Started](get-started).

### 操作說明

#### 第一節 — finalhe

1. 解壓 finalhe `.zip` 的檔案至您電腦的一個資料夾中
1. 將 `PSP2UPDAT.PUP` 檔案移至 `finalhe` 資料夾中
1. 在您的電腦中啟動 finalhe
  + 如果您使用的是 Windows 電腦且系統跳出防火牆警示，請允許 finalhe 連接至網路
1. 於您的主機中啟動「內容管理員」
1. 選擇『複製內容』
1. 選擇『個人電腦』
1. 選擇您想要連接至 finalhe 的方式
  + 如果您被要求登入至 PlayStation Network，請登入
  + 如果您並沒有一隻 PlayStation Network 的帳號，請創立一隻
1. 如提示時，選擇／註冊您的電腦
  + 若有提示更新的警示，請重新啟動您的主機並再試一次
  + 若您仍遇到該警示，於系統設定中啟用飛航模式，重啟主機並再試一次
  + 若您*仍*遇到該警示，請將飛航模式關閉並遵守[阻擋更新](blocking-updates)中的步驟，重啟並再試一次
  + 如果您的 Windows 電腦偵測不到您主機的 USB 連線的話，請安裝 [QcmaDriver_winusb](https://github.com/soarqin/finalhe/releases/download/v1.3/QcmaDriver_winusb.exe) 並再試一次
1. finalhe 此時應在您的畫面上顯示更新步驟

#### 第二節 — 更新韌體

1. 啟動設定
1. 移動至 `系統更新` -> `連接至個人電腦進行更新`
1. Ensure the update message "Version 3.65" is displayed, depending on which version you are attempting to update to
  + 如果顯示的不是此訊息，請立刻停止更新並重新檢查您是否漏了哪些步驟
1. 依循所示步驟以更新您的主機
  + 當更新完畢後，您的主機將自動重啟
