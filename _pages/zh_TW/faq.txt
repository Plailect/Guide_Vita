---
title: "常見問題 FAQ"
---

<a name="faq_latestfw" />**Q:** *我目前裝的是最新版的系統，這樣還可以改機嗎？*    
**A:** 可以的！

<a name="faq_downgrade" />**Q:** *我能降級我的 Vita 系統嗎？*    
**A:** 不行。

<a name="faq_update" />**Q:** *在安裝完自製韌體後，安裝官方更新是否安全？*    
**A:** 不！ 您必須確保您在[完成安裝](finalizing-setup)最後步驟中設置好防範未來的更新的措施。

<a name="faq_updates" />**Q:** *Sony 已經停產 Vita 了，那這樣不就代表未來不會再有系統更新了嗎？*    
**A:** 不一定。 系統版本 3.68 是在上一個更新釋出後的六個月後推出的（當時就已經假定不會再有新的更新了）。

<a name="faq_risky" />**Q:** *改機風險多高？*    
**A:** 除非您忽略本教學所有的警示，*基本上*變磚是不可能的。

<a name="faq_homebrew" />**Q:** *我可以透過本教學跑超讚的自製程式跟模擬器嗎？*    
**A:** 可以的！ 透過本教學您將可以在您的主機中運行任何社群所製的自製軟體。

<a name="faq_support" />**Q:** *如果我遇到問題該怎麼辦？*    
**A:** 若需支援，請見 [Discord 上的 Vita Hacking](https://discord.gg/JXEKeg6) 伺服器。

<a name="faq_nopc" />**Q:** *沒電腦也可以完成本教學嗎？*    
**A:** 不！ 您必須要有一台電腦才能完成本教學。
