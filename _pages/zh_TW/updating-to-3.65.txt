### 必讀事項

If you have already hacked your PS Vita (TV) before on 3.60, follow [Updating to 3.65 (HENkaku)](updating-to-3.65-(henkaku)) before continuing.
{: .notice--warning}

hencore 漏洞僅與韌體版本 3.65、3.67、3.68 相容。 因此，舊版的韌體必須先透過第三方伺服器更新才能使用本漏洞。

To do this, we configure your network's DNS settings to use a custom update server which will always tell your device that 3.65 is the latest update available. This has the effect of blocking any firmware versions above 3.65 from being installed when the device attempts to perform an update.

While the h-encore exploit itself is compatible with firmware versions 3.67 and 3.68, we will be updating to firmware version 3.65 as it is the latest version which still supports the Ensō boot-time exploit.

### 必備項目

* 在您的 PS Vita (TV) 可用的網路連線

### 操作說明

#### Section I - DNS Configuration

Note that these DNS servers will need to be set on each network you connect your device to for it to block updates!
{: .notice--warning}

1. 啟動設定
1. 移動至 `網路` -> `Wi-Fi 設定`
  + 若您尚未連上網，請連接到 Wi-Fi 網路
1. 選擇您目前所連結的網路
  + 已連上的網路旁邊會有一個綠色點點
1. 選擇『詳細設定』
1. 將『DNS 設定』改為『手動』
1. 將『主要 DNS』改為 `23.96.6.207`
1. 將『次要 DNS』留空
1. 確保您已將『Proxy 伺服器』設定至『不使用』
1. 回到主選單
1. Navigate to `System` -> `Auto-Start Settings`
1. Uncheck "Download Update File for System Software"
1. 回到主選單

#### 第二節 — 更新至 3.65

1. 移動至 `系統更新` -> `使用 Wi-Fi／行動網路更新`
1. 確保更新訊息為『3.65 (変革 Compatible)』
  + 如果顯示的不是此訊息，請立刻停止更新並重新檢查您是否漏了哪些步驟
1. 依循所示步驟以更新至 3.65
  + 當更新完畢後，您的主機將自動重啟
